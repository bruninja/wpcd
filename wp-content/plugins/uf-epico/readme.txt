== Installation ==

Como instalar o plugin?

- Se você acabou de instalar o WordPress, recomendamos instalar o tema primeiro. Para isso, baixe o tema e o plugin Épico na sua [área de clientes do Épico](https://minha.uberfacil.com/wp-login.php?redirect_to=http%3A%2F%2Fminha.uberfacil.com%2Farea-exclusiva-2). Por seguranca, baixe sempre os arquivos a partir da sua área de clientes.
- Instale o arquivo do tema em Aparência > Temas > Botão "Adicionar novo", dentro do painel administrativo do seu WordPress.
- Se já tiver instalado o tema anteriormente, instale o plugin em Plugins > Adicionar Novo.
- Para mais informações, consulte os tutoriais da sua [área de clientes](https://minha.uberfacil.com/wp-login.php?redirect_to=http%3A%2F%2Fminha.uberfacil.com%2Fdocs) para saber como utilizar todos os recursos do plugin.
